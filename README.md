# 📝 Todo List React

Aplicação de lista de tarefas feita com React + TypeScript.

## 🚀 Tecnologias

- React
- TypeScript
- TailwindCSS
- Phosphor Icons

## ✅ Funcionalidades

- Adicionar tarefas
- Editar tarefas
- Marcar tarefa como concluída
- Deletar tarefas
- Armazenamento no LocalStorage
- Modal de edição

## 📦 Instalação

```bash
cd todo-react
npm install
npm run dev
